Sourceforge offers a very slow connection for downloading these files by HTTP. If you have a SourceForge account you can get the files much faster via scp. On a posix system you should be able to do

scp username@quantlib.org:/home/groups/q/qu/quantlib/htdocs/prerelease/* .

On Windows, if you have PuTTY installed, you can do the following from a DOS prompt:

"C:\Program Files\PuTTY\pscp.exe" "username@quantlib.org:/home/groups/q/qu/quantlib/htdocs/prerelease/*" "C:\projects"
