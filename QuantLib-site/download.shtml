Download Page

The latest QuantLib release is the 0.3.12 version (March 2005.)

Debian users will find version 0.3.12 of the different QuantLib packages in the "unstable" (a.k.a. "Sid") distribution; see the Debian site for a list of available packages.

For Mac OS X users, Fink packages will be available in the "10.3/unstable" tree; for a list of available QuantLib-related files see the Package Database. To enable the "unstable" tree, please read the Fink faq. Be sure to run "fink selfupdate" to obtain the latest package definitions, and then run "fink install quantlib" to install the main library.

Source packages for other platforms are available from the SourceForge download page in a number of different formats. This also applies to eager Debian or Mac OS X users not wanting to wait for the library to be packaged, or to advanced users wanting to configure the library with non-default settings. Compiling QuantLib requires a working Boost installation; Boost 1.31 or later is needed, while Boost 1.33.1 is suggested.
